Geo-Eyes Coordinate Finder

How It Works:
1. User Input:
The surveyor inputs the easting and northing coordinates of the point they are interested in.
2. Database Search:
The application searches through a pre-uploaded database of existing coordinates (pillars)
within Oyo State to find the three closest points to the input coordinate.
3. Distance Calculation:
The application calculates the Euclidean distance between the input coordinate and each point
in the database to determine proximity.
4. Results Display:
The application displays the three nearest points along with their coordinates and the calculated
distances.

2. Instantly receive the three closest existing pillars, along with their distances.
3. Use this information to connect the new point to the existing survey network, ensuring accuracy and efficiency in their measurements.
